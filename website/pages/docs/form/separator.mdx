Box is the most abstract component on top of which all other Purity UI Dashboard
components are built. By default, it renders a `div` element.

<br />

**The Box component is useful because it helps with 3 common use cases:**

- Create responsive layouts with ease.
- Provide a shorthand way to pass styles via props (`bg` instead of
  `backgroundColor`).
- Compose new component and allow for override using the `as` prop.

## Import

```js
import { Separator } from "components/Separator/Separator"
```

### Usage

```jsx
// Sample card from Airbnb

function AirbnbExample() {
  const property = {
    imageUrl: "https://bit.ly/2Z4KKcF",
    imageAlt: "Rear view of modern home with pool",

    title: "Modern home in city center in the heart of historic Los Angeles",
    formattedPrice: "$1,900.00",

    rating: 4,
  }

  return (
    <Box>
      <Text align="center" mb="20px">
        This is on above the Separator
      </Text>
      <Flex
        h="1px"
        w="100%"
        bg="linear-gradient(90deg, rgba(224, 225, 226, 0) 0%, #E0E1E2 49.52%, rgba(224, 225, 226, 0) 100%)"
        mb="20px"
      ></Flex>
      <Text align="center">This is on below the Separator</Text>
    </Box>
  )
}
```
